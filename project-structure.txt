## Key Directories

| Directory | Purpose |
|-----------|---------|
| `src/pat/` | Post-Quantum Aggregate Transactions (Dilithium) |
| `src/wallet/pqwallet/` | Quantum-resistant wallet module |
| `src/crypto/` | Cryptographic primitives |
| `doc/wallet/` | Wallet documentation for auditors |
| `doc/specifications/` | Protocol specifications |
| `design-log/` | Engineering design decisions |

## Build Quick-Start

```bash
./autogen.sh
./configure --with-boost=/opt/homebrew/opt/boost
make -j$(nproc)
./src/soqucoind -testnet
```
